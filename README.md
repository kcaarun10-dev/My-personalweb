# Arun Regmi — Hybrid Web Platform 🚀

This repository hosts the unified digital presence of **Arun Regmi**, accessible at [arunregmi.com.np](https://arunregmi.com.np). It combines a high-performance **Developer Portfolio** with the **ArunTech Blog** and a suite of **Online Tools**.

## 🏗️ Unified Architecture

- **Home Page (`/`)**: Modern developer portfolio with interactive typing effects and custom branding.
- **ArunTech Blog (`/blog`)**: SEO-optimized technical blog featuring smartphone leaks, AI news, and gadget reviews.
- **Online Tools (`/tools`)**: A collection of 17+ utility tools (Word counters, converters, QR generators).
- **Admin Dashboard (`/admin`)**: Secure Markdown CMS for managing blog posts and monitoring comments.

## 🛠️ Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS + Custom Portfolio CSS
- **Database**: Firebase (Firestore, Authentication, Storage)
- **Monetization**: Google AdSense + Universal Ad Unit (Supports Popunders & Social Bars)
- **Content**: Markdown-driven (React-Markdown, Remark GFM)

## 🚀 Deployment

This project is configured for seamless deployment on **Vercel** or **GitHub Pages** (via Vercel for dynamic routes).

1. **Clone & Install**:
   ```bash
   npm install
   ```

2. **Run Locally**:
   ```bash
   npm run dev
   ```

3. **Deploy**:
   Push to the `main` branch of your GitHub repository at `kcaarun10-dev/My-personalweb`.
